# SQL Scripts

This folder contains the complete SQL script used for the Customer Churn Analysis project.

Customer_Churn_Analysis.sql
   - Creates the database
   - Creates the customer_churn table
   - Creates 20 supporting tables
   - Imports and manages banking customer churn data
   - Contains 35 business-focused SQL queries
   - Includes views, joins, subqueries, and window functions

SQL Concepts Used:
- SELECT Statements
- Aggregate Functions
- GROUP BY
- HAVING
- ORDER BY
- CASE Statements
- Subqueries
- INNER JOIN
- LEFT JOIN
- Window Functions
- Views

Project Features:
- Customer Churn Analysis

- Branch Performance Analysis
- Product Performance Analysis
- Revenue Analysis

- Executive Reporting
